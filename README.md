# Frila — App Mobile

Aplicativo de serviços freelance desenvolvido com React Native + Expo. Utiliza SQLite local (expo-sqlite) para persistência de dados, sem necessidade de servidor externo.

---

## Pré-requisitos

- [Node.js](https://nodejs.org/) instalado
- [Expo Go](https://expo.dev/go) instalado no celular

---

## Instalação

```bash
npm install
```

---

## Executar o projeto

```bash
npx expo start
```

Após iniciar, escaneie o QR Code com o **Expo Go**
